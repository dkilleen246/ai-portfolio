# Evaluations

Structured AI tool evaluation reports using professional analyst methodology — rubric-based scoring, multi-model comparisons, and evidence-backed recommendations.

## Structure

Each evaluation includes:
- **Objective** — what's being evaluated and why
- **Methodology** — evaluation criteria and scoring rubric
- **Test Cases** — specific prompts or tasks used to test each tool
- **Findings** — results per tool, per criterion
- **Recommendation** — which tool wins for this use case and why
