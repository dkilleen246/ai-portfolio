# Prompt Library

A growing collection of documented prompt engineering examples — each showing the problem, the iteration process, and the final working solution.

## Structure

Each entry includes:
- **Use Case** — what problem the prompt solves
- **First Attempt** — the initial prompt and why it fell short
- **Iterations** — what changed and why
- **Final Prompt** — the reliable, reusable version
- **Result** — what the output looks like and why it works
- **Model** — which AI model was used (Grok / Claude)
